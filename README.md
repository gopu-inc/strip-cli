# 🎬 STRIP Streaming

**STRIP** est une plateforme de streaming vidéo et messagerie temps réel, construite avec FastAPI et Textual.

## ✨ Fonctionnalités

- 📹 **Streaming vidéo** : Upload et lecture de vidéos
- 💬 **Messagerie temps réel** : Chat avec WebSockets
- 🔴 **Lives** : Streaming en direct
- 🎵 **STRIP Sounds** : Bibliothèque audio
- ✅ **Système de vérification** : Badge sécurisé
- 📱 **Interface TUI** : Terminal UI avec Textual

## 🚀 Installation

```bash
store install com.strip.streaming
```

Ou manuellement :

```bash
git clone https://github.com/gopu-inc/STRIP
cd STRIP
pip install -r requirements.txt
pip install -e .
```

📦 Dépendances principales

· FastAPI : API backend
· Textual : Interface terminal
· WebSockets : Messagerie temps réel
· Cloudinary : Stockage vidéo
· PostgreSQL : Base de données


🎮 Utilisation

```bash
# Démarrer le strip
strip
```
